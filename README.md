# 🎬 Netflix Homepage UI Clone

A responsive Netflix Homepage UI Clone built using **HTML, CSS, and JavaScript** as part of a UI Clone Challenge.

> **Disclaimer:** This project is created for educational purposes only. It is not affiliated with or endorsed by Netflix. No authentication or backend functionality is implemented.

---

## 📌 Project Overview

This project recreates the look and feel of the Netflix landing page using only front-end technologies. It focuses on responsive design, modern layouts, and interactive UI elements.

---

## ✨ Features

- Responsive Navigation Bar
- Hero Banner Section
- Feature Sections
- FAQ Accordion
- Responsive Design
- Smooth Scrolling
- Modern UI
- Mobile Friendly
- Interactive Buttons

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
